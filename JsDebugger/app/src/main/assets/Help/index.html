ヘルプページです。

このアプリは武田家のうーたんがつくりました
このアプリではデバッガの入力欄にJavaScriptのコードを入力することでアプリ上でそれを実行できます
またエラーがあれば表示される機能付きです
Web版もあります→JsDebugger

！！！アプリ限定要素！！！

アプリでのみログ要素が実装されてます

ログを出力したいところで
android.addLog(STRING_LOG);
とするとログが出力されます(STRING_LOGには出力する文字列をいれてください)
出力されたログは右上のメニューにあるLogから見れます
ログはデバッガに移動する際に初期化されます

お気に入り機能も実装されてます

残しておきたいコードを入力後
Debugボタンの横のFavoriteボタンを押してください
するとダイアログがでるので保存するコードの名前をいれてください
保存後メニューからFavoriteを選ぶとコードが表示されています
名前が書いてあるところを押すと一覧が出るので、見たいものを選んでください
消すときは下のDeleteを使ってください
また、Copyを押すとそのコードをコピーできます

共有機能も実装されてます

Debuggerの画面ならコード入力部分の下、
Favoriteの画面なら右上のShareボタンを押すとコードを共有できます
Debuggerの画面のときはダイアログがでるので指示に従ってください
共有先はSNSやテキストエディタなどが選べます
また、他のアプリ(テキストエディタ)などから共有された場合コードを受け取ることもできます
